HTML and CSS Reference

HTML Element Tags
Tag Name Description
<body> Body Any visible content should be placed within the opening and closing body tags.
<h1> - <h6> Headings Headings and sub-headings, h1 to h6 tags, are used to provide titles for sections of content.
<p> - <span> - <div> Paragraph, Span, and div p, span, and div tags specify text or blocks.
<em> - <strong> Emphasized and Strong The em and strong tags are used to emphasize text.
<br> Breaks Line breaks are created with the br tag.
<ol> - <ul> Ordered and Unordered lists Ordered lists (ol) are numbered and unordered lists (ul) are bulleted.
<img> - <video> Images and Videos Images (img) and Videos (video) can be added by linking to an existing source.

Table Attributes
Attribute Name Description
colspan Column Span Defines how many columns a td element should span.
rowspan Row Span Defines how many rows a td element should span.
